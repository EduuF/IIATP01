Na pasta raiz do projeto também contém 2 scripts, um para automatizar a compilação em ambiente Windows e outro para Linux

- Windows: 
O arquivo TP1.bat na pasta raiz do projeto é um script que compila o projeto automaticamente em ambientes Windows, para compilar basta digitar o comando:
TP1 {Algoritmo} {número de elementos no vetor} {vetor} {PRINT}

Exemplo de comando: TP1 A 5 5 2 3 1 4 PRINT

- Linux: 
O arquivo TP1 na pasta raiz do projeto é um script bash que compila o projeto automaticamente em ambientes Linux, para compilar basta digitar o comando:
./TP1 {Algoritmo} {número de elementos no vetor} {vetor} {PRINT}

Exemplo de comando: ./TP1 A 5 5 2 3 1 4 PRINT
